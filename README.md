# zoomsdk-swift-demo

An example app using Zoom SDK for iOS and Swift

# Before running this example, configure your App Key and App Secret.

Copy

> ZoomDemo/Keys.example.swift

to

> ZoomDemo/Keys.swift

, then edit the file and put your keys.
